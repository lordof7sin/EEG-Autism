# EEG-Autism

EEG-Autism es un proyecto de desarrollo de un electroencefalógrafo diseñado para ayudar en la identificación de emociones en personas con trastorno del espectro autista (TEA). El sistema tiene como objetivo proporcionar una herramienta de apoyo basada en señales cerebrales para mejorar la comprensión emocional y fomentar una mejor comunicación.

## Descripción general

Este proyecto busca implementar un sistema completo de registro, procesamiento y análisis de señales EEG (electroencefalográficas) para determinar patrones emocionales en personas con TEA. La solución incluye tanto el diseño de hardware como el desarrollo de una base de datos y entorno de análisis digital.

## Tecnologías utilizadas

- Diseño de PCB: Eagle
- Base de datos: XAMPP (MySQL y Apache)
- Interfaz y backend: NetBeans (Java)
- Lenguajes principales: Java, SQL

## Cómo comenzar

### Requisitos previos

- NetBeans IDE
- XAMPP instalado (MySQL y Apache)
- Eagle para visualizar/modificar el diseño de la PCB

### Instalación

1. Clona este repositorio:
   git clone https://github.com/tu-usuario/EEG-Autism.git
2. Abre el proyecto en NetBeans.
3. Configura XAMPP y crea la base de datos usando los scripts disponibles en la carpeta /database/.
4. Corre Apache y MySQL desde el panel de XAMPP.
5. Ejecuta el proyecto desde NetBeans.

## Objetivos del sistema

- Capturar señales EEG en tiempo real.
- Clasificar emociones mediante análisis de ondas cerebrales.
- Registrar y visualizar resultados en una base de datos.
- Ser un sistema de bajo costo y fácil implementación para estudios con personas con TEA.
